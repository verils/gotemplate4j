# GoTemplate4J Documentation

Welcome to the comprehensive documentation for **gotemplate4j** - a Go template engine implementation for Java.

## What is GoTemplate4J?

GoTemplate4J brings the power and simplicity of Go's `text/template` package to Java applications. It enables you to:

- Use familiar Go template syntax in Java projects
- Generate dynamic text output from templates
- Work with Java objects, maps, and collections seamlessly
- Leverage built-in functions and create custom ones
- Maintain compatibility with existing Go templates
