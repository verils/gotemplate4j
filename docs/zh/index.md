# GoTemplate4J 文档

欢迎来到 **gotemplate4j** 的全面文档——一个用于 Java 的 Go 模板引擎实现。

## 什么是 GoTemplate4J？

GoTemplate4J 将 Go 的 `text/template` 包的强大和简洁带到了 Java 应用中。它让你能够：

- 在 Java 项目中使用熟悉的 Go 模板语法
- 从模板生成动态文本输出
- 无缝操作 Java 对象、Map 和集合
- 利用内置函数并创建自定义函数
- 保持与现有 Go 模板的兼容性

## 核心特性

✅ **Go 模板兼容** - 支持核心 Go `text/template` 语法  
✅ **Java 11 兼容** - 适用于 Java 11 及以上版本  
✅ **零依赖** - 纯 Java 实现  
✅ **丰富的函数库** - 18+ 内置函数  
✅ **自定义函数** - 简易的扩展机制  
✅ **类型安全** - 利用 Java 的类型系统  
✅ **充分测试** - 90%+ 代码覆盖率
